# 中文课程期末作业用论文 Typst 模板

满足以下要求：
- 题目（三号宋体，加粗，居中）
- 学号 姓名 院系（小四号宋体，加粗，居中）
- 摘  要：（顶格、五号宋体）
- 关键词：（顶格、五号宋体；3-5个，关键词用分号隔开）
- 一级标题：一、二、三、（四号，宋体加粗）
- 二级标题：（一）（二）（三）（小四，宋体加粗）
- 三级标题： 1.2.3.（五号，宋体）
- 正文内容：（五号，宋体）
- 正文行距：（1.5倍行距）
- 英文：默认统一用 Times New Roman 字体

### 字体

Typst 无法显示宋体的粗体。因此我们使用了支持的思源宋体作为替换。

为了能正常使用粗体，请安装 [Source Han Serif SC](https://github.com/adobe-fonts/source-han-serif)， 宋体（simsun）
